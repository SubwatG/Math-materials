Interactive Definition 1.3

Logistic Growth Function

ปรับค่า a, b, และ c เพื่อสังเกตว่าฟังก์ชันโลจิสติกเริ่มจากน้อย เพิ่มเร็วขึ้น แล้วค่อย ๆ เข้าใกล้ขอบเขตบนอย่างไร

บทนิยาม 1.3

กำหนดให้ a, b, c เป็นค่าคงตัวบวก โดยที่ b < 1 ฟังก์ชันโลจิสติกของการเพิ่มปริมาณ คือ

$$ f(x) = \frac{c}{1 + a b^x} \quad \text{หรือ} \quad f(x) = \frac{c}{1 + a e^{-kx}} $$

เมื่อ c คือขอบเขตของการเพิ่มปริมาณ (the limit to growth) และในกราฟนี้ใช้ความสัมพันธ์ $k = -\ln(b)$
